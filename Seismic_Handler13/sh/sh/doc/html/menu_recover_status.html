Menu entry Recover Status

Restores picks and analysis parameters from a previously saved binary file in the $SH_SCRATCH directory (with Final Parameters). This option allows to recover a previous event analysis. The binary file contains the event ID number. The desired file has to be selected using the file selection box. The difference to Recover Evt-File is that also the event ID number is recovered and a following call to Final Parameters overwrites the previous version of the binary status file and the evt-file. This way a data analysis can be continued.

Please note that the binary status files are not appropriate for permanent archiving since the binary format may change with every version upgrade of SHM. Binary status files of older SHM version cannot be used any more. For permanent archiving use the evt text files.
