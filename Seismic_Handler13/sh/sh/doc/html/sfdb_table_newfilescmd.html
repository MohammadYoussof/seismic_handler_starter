sfdb Table newfilescmd

  • id INTEGER,
  • m_cmd VARCHAR(255),
  • d_cmd VARCHAR(255),
  • PRIMARY KEY(id)

Description:

  • id: path ID; description see 'pathid' in table sftab
  • m_cmd: command to find new data files in path 'id' wich are modified within the last N minutes. In the resulting command string the string '__TIME__' has to be replaced by the number N. Example: ssh rtpdserv "(cd /home/sysop/seiscomp_2_5/seiscomp/acquisition/archive; find . -type f -mmin -__TIME__ -print)"
  • d_cmd: command to find new data files in path 'id' wich are modified within the last M days. In the resulting command string the string '__TIME__' has to be replaced by the number M. Example: ssh ersn23 "(chdir /r06p4/arch_bh; find . -type f -mtime -__TIME__ -print)"
