Steps to analyse local events

  • read in appropriate data window
  • choose filter which provides good signal/noise ratio
  • pick phases Pg, Sg, Pn, Sn if possible
  • call menu entry LocSAT and locate event
  • check residuals, correct phase readings, relocate if necessary
  • choose Wood-Anderson filter
  • select menu entry ml and select maximum amplitude window on one horizontal component for each station
  • select menu entry Final Parameters to save results to file
