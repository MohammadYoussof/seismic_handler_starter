Menu entry Read Q

Reads in Q data files. Q-files is a data format used in the command line version of SeismicHandler (SH). Read the SH documentation $SH_ROOT/doc/sh.ps to learn details about the Q-file format. Each data set consists of two files (file.QHD and file.QBN), a header file and a binary data file. A file selection box pops up to let the user select the file. Please select the .QBN file. The default path in the selection box may be changed in Configure (parameter number 70). After selecting the file, all traces of the file are read in. By default all previously read traces are deleted before reading a new Q file. To keep traces when reading new ones switch on the Keep Traces button in the Read dialog box (this is not straightforward, I admit).
