New version SHM-2.4 now available with major improvements in configuration. Please read migrating to 2.4 about improvements and how to change to this version.

SHM (Seismic Handler Motif) is an interactive analysis program for preferably continuous waveform data.  It was developed at the Seismological Observatory Gräfenberg and is in use here for daily routine analysis of local and global seismic events. Read here for an overview on the program features.
