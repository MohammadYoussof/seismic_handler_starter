Menu entry Recover Evt File

Restores picks and analysis parameters from a previously saved evt file (with Final Parameters). This option allows to recover a previous event analysis. The desired file has to be selected using the file selection box. The difference to Recover Status is that the event ID number is not recovered and a following call to Final Parameters creates a new event ID. Due to the flexible file format of the evt-files, even evt-files from old versions of SHM can be recovered.
