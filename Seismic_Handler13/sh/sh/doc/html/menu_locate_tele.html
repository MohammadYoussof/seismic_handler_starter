Menu entry Locate Tele

Determines source location using teleseismic phases with plane wave approximation. It takes the slowness value from the analysis parameter box and determines an epicentral distance by using travel time tables of the phase given in the phase dialog box. This distance together with the back-azimuth from the analysis parameter box gives the source location by connecting it to a reference station. It uses calibrated values of slowness and back-azimuth if available. The source location, the origin time and the epicentral distance to the reference station is written to the analysis parameter box.

The travel time tables used in this option are located in the $SH_INPUTS directory and have a filename extension of TTT.
