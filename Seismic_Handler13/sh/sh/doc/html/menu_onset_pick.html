Menu entry Onset Pick and Menu entry Onset Pick Setup

Implementation of an onset automatic phase picker. The algorithm is described in An Automatic Phase Picker for Local and Teleseismic Events by M. Baer and U. Kradolfer, BSSA, Vol. 77, No. 4, pp 1437-1445, Aug 1987. A time window must be selected using the right mouse button on any trace of the display. After selecting Onset Pick the above algorithm is applied to the same time window for all traces of the display. Any resulting picks are marked, the phase name is taken from the phase dialog box. This algorithm is controled by three parameters Threshold, Duration and Breaklength. The parameters are accessable via the menu entry Onset Pick Setup under menu title Specials.

This option has not been used very often, so please be prepared to find bugs.
