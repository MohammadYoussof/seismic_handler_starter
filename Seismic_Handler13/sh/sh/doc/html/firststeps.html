First steps

After the program package has been installed correctly and all parameters are set properly you may start to work with SHM.

Starting SHM

Move a terminal window to the lower left corner of the screen (to see possible messages of the program later on) and enter SHM. All windows needed will pop up automatically.

Reading in GSE formatted data

Select the menu entry Read GSE and select one of the GSE-files offered. If the installation script was used, the paths to the example GSE files should be properly set. You may read any other GSE-file available on your filesystem with this option.

Reading in MiniSEED data

For reading your own data with this option you need to configure the dialog box correctly. Please read about preparing your MiniSEED files and inserting your own stations. If configured, select menu entry Read. This opens a dialog box. Select appropriate buttons for stations, data channel (e.g. BH) and component(s). Choose date and time using the arrow buttons above and below the time field. Click on the Read Again button.

Filter data

Select the Filter menu entry. Select desired short-period or long-period simulation filters if you have read in broadband data. Click on OK. Change filters just by repeating these steps here. If you see zero lines read about the proper setup of simulation filters.

Change time resolution

Navigate in time using the right mouse button and the entries of the Window menu title. Please note that you cannot see more data than that you read in using the Read entry.

Change amplitudes

Change amplitudes of traces using the amplitude ruler within the parameter box left to the main display window. Amplitudes can be changed in discrete nonlinear steps.

Leaving program

Exit the program by calling the menu entries Cancel Parameters and then Quit.
