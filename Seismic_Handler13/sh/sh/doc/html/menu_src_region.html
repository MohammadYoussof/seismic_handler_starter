Menu entry Src Region

Computes location by adding distance and back-azimuth from the analysis parameter box to the location of the reference station. Please check for a proper setup of the reference station (menu entry Ref. Station). Returns also name of the region. Uses sphere geometrics for this computation, correction for ellipticity is not applied, since most travel time tables are also created without ellipticity.
