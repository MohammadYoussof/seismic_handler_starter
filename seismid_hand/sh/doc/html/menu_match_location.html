Menu entry Locations Matching

Obsolete entry. Please use Identify Phase instead.

Pick a phase and determine its slowness and azimuth using array methods. Then after clicking this entry SHM searches the NEIC finger list (quake@ghtftp.cr.usgs.gov) for matching events. The best fitting event within some maximum tolerance limits is copied to the parameter box.
