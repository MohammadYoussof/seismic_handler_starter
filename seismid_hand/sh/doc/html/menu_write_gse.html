Menu entry Write GSE1.0

Write the current content of the display window into a GSE1.0 formatted data file. The filename is generated automatically. Output path is the current working directory. Please note that GSE formatted files take integer sample values only. If you want to write processed data (e.g. filtered data) be aware that the samples are truncated resulting in possibly unusable data files. Preferably write unfiltered data to GSE files.
