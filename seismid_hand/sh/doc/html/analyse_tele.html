Steps to analyse teleseismic events

  • read in appropriate data window
  • choose filter which provides good signal/noise ratio (WWSSN-SP)
  • determine slowness and azimuth of incoming wave (a) using manually picked first arrivals, then call Plane Wave, (b) using first visible minimum/maximum automatically picked, then call Plane Wave, or (c) using FK algorithm.
  • check resulting slowness and azimuth using the Beam option of SHM, if necessary correct phase readings
  • pick first arrival phase (P, PKP) on reference station, add phase readings on other stations if desired
  • calibrate slowness and azimuth if calibration vectors are available (e.g. for GRF) using menu entry Calibration
  • call menu entry Locate Tele and locate event
  • if depth phases are visible, pick them and determine focal depth (menu entry Depth). Relocate event (Locate Tele) with new depth.
  • read maximum amplitude on first arrivals picked (WWSSN-SP simulation filter, menu entry Ampl & Period auto Peak)
  • determine magnitudes mb (menu entry mb)
  • reread event in a longer time window to see possible surface waves. Use only vertical component of reference station
  • choose SRO-LP simulation filter, adjust cutoff time to at least 5 min.
  • use menu entry Surface Ampl automatic and select window of surface wave train on vertical component to determine amplitude and period of surface waves
  • determine Ms using menu entry MS Prague
  • select menu entry Final Parameters to save results to file
