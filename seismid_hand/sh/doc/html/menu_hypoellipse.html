Menue entry Hypoellipse

Calls external program Hypoellipse for locating local (and regional) events. The external program is not part of the distribution. It must be provided separately under the name Hypoe in $SH_UTIL. A SunOS version is available on our ftp server. This option is subject to change in a future release of SHM. It will be used to provide a more general interface to external location programs.
