Menu entry Correlation Pick

Create automatic phase picks using a correlation algorithm. Choose a reference trace and pick the phase onset manually on this trace. Then select a signal window including some pre-onset time using the right mouse button drag window. After activating Correlation Pick all traces of the display are correlated with the selected time window on the reference trace. At the position with maximum correlation automatic phases are picked with the same relative position within the selected window. This correlation pick option works well with teleseismic phases with high signal coherency.
