Menu entry Align

Shifts all traces so that the phase with name given in the phase dialog box is aligned in time. Computes a sum trace over all traces on display. Useful to get an enhanced signal to noise ratio on the sum trace for amplitude measurement. Please note that there is no check whether the phases have been properly selected (in opposite to the Beam command).

This entry is organized as a toggle button. Selecting it twice restores the unshifted traces and removes the sum trace.
