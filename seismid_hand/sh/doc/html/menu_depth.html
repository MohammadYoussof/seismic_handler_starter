Menu entry Depth

Determination of source depth of teleseismic events using depth phases. SHM searches for pairs of depth phases and corresponding main phases. SHM needs to know which depth phases should be processed if they are picked. This list of depth phases has to be specified in the Configure option (parameter number 1). Changing the existing parameter entry means to retype the complete list of phases including or excluding your changes. Please check for typos and do not use blanks, just separate the phases by commas, there is no check implemented ! Phases not listed here will be ignored in the depth determination. Selecting the Depth entry makes SHM scan all traces for pairs of known phases. Each pair is assigned a depth by fitting the time distance between the phase pair using the travel time tables in $SH_INPUTS. All depths found are averaged, the resulting mean depth is written to the analysis parameter box.
