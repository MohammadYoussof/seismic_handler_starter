Menu entry External Loc

After a location (latititude and longitude) of an external source has been written to the analysis parameter box manually, this option computes the epicentral distance to the reference station and the corresponding P slowness and puts the results into the analysis parameter box. Please check the proper setting of the reference station (menu entry Ref. Station). The distance computation uses the correction for ellipticity of the earth.
